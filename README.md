# Ansible Basic install

## Setup

```
make setup
```

## Run with

```
make load
```

This will install those applications:

- zsh
- docker
- git
- php
- npm / yarn

## Applications :

This also install app :

### PhPStorm :

Follow the rules in the /opt/Install-Linux-tar.txt to install PhPStorm

### Slack / Chrome / VSCode

Should run without any configuration
